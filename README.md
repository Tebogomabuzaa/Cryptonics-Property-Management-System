# Cryptonics Property Management System

## Overview
Cryptonics Property Management System is a Phase 1 academic project proposal for a modern property management solution designed to improve how international property portfolios are managed.

The project focuses on solving the operational challenges faced by Atlas Premier Properties, including manual rental settlement calculations, slow cross-border payments, fragmented records, and limited reporting capabilities. The proposed system aims to streamline property administration, tenant vetting, financial settlements, and management reporting through a secure web-based platform.

## Problem Statement
Atlas Premier Properties currently relies on a legacy system made up of spreadsheets, email, and manual processes. This creates delays, increases the risk of human error, and makes it difficult to manage tenants, owners, and settlements efficiently.

Key issues identified in the proposal include:
- Manual split-payment calculations
- Slow international payments
- Delays in tenant vetting
- Fragmented data storage
- Limited management reporting
- Weak support for international clients
- Poor document and record management

## Proposed Solution
The proposed Cryptonics system will act as a centralized property management portal with the ability to:
- Manage properties, owners, tenants, managers, lease agreements, and settlements
- Automate tenant vetting using external verification services
- Generate cryptocurrency-based invoice/payment links
- Calculate and distribute split payments
- Provide graphical management reports and business insights
- Support help functionality for users

## Scope
The project scope includes:
- CRUD functionality for core entities
- Lease and settlement management
- Automated financial settlement logic
- Reporting dashboards for management
- Help and support functionality

Out of scope / constraints include:
- Building a private blockchain from scratch
- Storing sensitive passport data directly
- Using a third-party KYC integration for compliance purposes

## Objectives
The main goals of the project are to:
- Improve business processes through automated settlement calculations
- Improve communication and payment speed for international owners
- Improve decision-making through real-time reporting and dashboards

## Opportunities for Future Improvement
Possible future enhancements include:
- Smart contract integration
- Automated tax report generation
- Exchange rate tracking for crypto-to-Rand settlements
- AI chatbot support for tenant and owner assistance

## Project Timeline
The proposal follows a structured SDLC approach covering:
- Scope definition
- Problem analysis
- Requirements analysis
- Logical design
- Decision analysis
- Physical design and integration
- Construction and testing
